# Heldenwerft
Social Learning Platform

## Install
```
git clone https://github.com/phusa/heldenwerft
```

### Frontend
```
cd app
npm install
npm run dev
```
### Backend
```
cd cms
npm install
npm run strapi dev
```

## Tech stack
* Strapi Headless CMS
* Svelte + Sveltekit
* DaisyUI + Tailwind
